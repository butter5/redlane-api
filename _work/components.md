## Backend API Components

### API Structure
This project uses Laravel 11's API-first approach with the following structure:

- **Controllers:** Handle HTTP requests and responses
- **Resources:** Transform models into JSON responses
- **Requests:** Validate incoming API requests
- **Middleware:** Process requests before reaching controllers

### API Documentation Tool
- **Scribe:** Auto-generates API documentation from annotations
- Accessible at `/docs` endpoint
- Generates Postman collections
- Interactive API testing interface
